|                           | Translational    | Connection                    | Rotational | 
| ------------------------- | ----------------- | ----------------------------- | ------- | 
| Base Quantities           | $x,y,\Delta{x},\Delta{y},t$  |     | $\Theta, \Delta{\theta}, t$    |
| Coordinates               |   |     |  |
| Velocity                  | $v$  | $v=rw$                        | $\omega$  |

| Big 3 Equations of Motion | [[Motion]] |                               | [[Radial Motion]] |
